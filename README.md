# Description:

Quick Browser TV is a web browser designed for Fire TV and Android TV devices, with a navigation experience optimized for remote control use.

The app provides a TV-friendly browser interface with an on-screen pointer, direct address bar navigation, favorites management, quick actions, and support for downloading compatible files.

Main features:
- Web browsing optimized for TV devices
- Remote-friendly on-screen pointer
- Address bar with direct URL input
- Favorites saving and management
- Quick menu for common browser actions
- Downloads section for saved files
- Big-screen friendly interface

Quick Browser TV is designed to make browsing easier on television devices without requiring touch input.

The app allows users to:
- Open websites directly from the address bar
- Search the web quickly
- Save and remove favorite pages
- Access a full menu and quick action panel
- Download supported files for local use
- Manage previously downloaded files

The interface is designed for Fire TV and Android TV environments, with a simple layout and practical navigation controls for remote usage.
